# Employee Management MVC

An Employee Management System built using ASP.NET Core MVC.

## Features

- Employee CRUD Operations
- Search Employees
- Pagination
- Dashboard
  - Total Employees
  - Average Salary
  - Department Count
- ASP.NET Core Identity Authentication
- Responsive Bootstrap UI
- Repository Pattern
- Service Layer
- Entity Framework Core
- SQL Server

## Technologies Used

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap 5
- ASP.NET Core Identity

## Project Structure

- Controllers
- Models
- Views
- Services
- Repository
- Data
- wwwroot

## How to Run

1. Clone the repository
2. Open in Visual Studio
3. Update Connection String
4. Run the project
